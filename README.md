# Vista Roofing - Professional Website

A modern, professional website for Vista Roofing, a premier roofing contractor serving St. George, Utah and surrounding areas.

## 🎨 Design Features

### Modern Design System
- **Custom CSS Properties**: Comprehensive design tokens for colors, typography, spacing, and shadows
- **Professional Color Palette**: Blue primary colors with orange accents, perfectly suited for roofing industry
- **Typography**: Inter for body text, Poppins for headings - clean and professional
- **Responsive Grid System**: CSS Grid with auto-fit columns for perfect responsiveness
- **Smooth Animations**: Fade-in animations and hover effects for enhanced user experience

### Visual Excellence
- **Hero Section**: Full-screen hero with professional roofing background image
- **Service Cards**: Elegant cards with icons and hover effects
- **Testimonials**: Customer reviews with photos and star ratings
- **Contact Form**: Modern form with real-time validation and beautiful styling
- **Mobile-First**: Fully responsive design that looks great on all devices

## 🔧 Setup Instructions

### Requirements
- Web server with PHP support
- Email functionality (SMTP or mail() function)

### Installation
1. Upload all files to your web server
2. Ensure PHP is enabled
3. Create logs directory with write permissions
4. Test the contact form functionality
5. Update contact information as needed

### Configuration
- Update email recipient in `assets/php/contact.php`
- Modify contact information in HTML files
- Customize colors and branding in CSS custom properties
- Add your own images to the assets/images directory
